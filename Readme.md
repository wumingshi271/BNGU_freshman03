# OpenCV最小框选与多边形拟合（选做题）

该项目运用颜色分割，实现了下面视频的蓝牙耳机给框选出来(这边只展示视频中的一帧)

![原始视频](Screenshot%202025-10-10%20164420.png)

## 环境配置

### 配置opencv

在终端输入以下指令

`pip install opencv-python`

## 实现思路

通过提取粉色的hsv掩码，将图片中的粉色单独提取出来

掩码如下所示：

| hsv | 下界 | 上界  |
| :-: |:--:|:---:|
|  h  | 0  | 255 |
|  s  | 0  | 255 |
|  v  | 0  | 60  |

然后将视频里的蓝牙耳机框选出来
